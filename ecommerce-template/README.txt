# Fashion Store - HTML E‑commerce Template

A clean, static front-end e‑commerce homepage with mega menu, hero, categories, products and reviews.

## What's included
- index.html
- assets/css/style.css
- assets/js/script.js
- assets/img/ (place images here)

## Features
- Sticky header + search
- Hover mega menu (MEN, WOMEN, KIDS, HOME, BEAUTY, GENZ)
- Hero banner
- Category grid
- Product cards (dummy data via JS)
- Reviews section
- Responsive (tablet/mobile)

## How to use
1. Open `index.html` in your browser.
2. Edit brand text in the header and hero in `index.html`.
3. Change colors/spacing in `assets/css/style.css` (uses CSS variables and utilities).
4. Replace emoji placeholders or add real product images in `assets/img/` and update the HTML/JS.

## Notes
- Tailwind CDN is included for convenience. You can remove it if you prefer only custom CSS.
- No external editor footprints. All Canva/Cloudflare traces have been removed.
